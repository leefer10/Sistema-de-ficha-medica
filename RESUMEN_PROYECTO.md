# PROYECTOS

## Sistema de Hermanos Para su Salud

**Sin Fines de Lucro** | 2024 - 2025

Aplicación web y móvil para gestión integral de salud. Permite a usuarios gestionar medicamentos, citas médicas y digitalizar documentos clínicos mediante OCR/IA.

- **Frontend Completo**: React 18 + Next.js 14 con 8 componentes principales, 22 validaciones y design system personalizado.
- **Gestión de Datos**: CRUD completo para medicamentos y citas con recordatorios automáticos, tracking de consumo y notificaciones en tiempo real.
- **Autenticación Segura**: JWT con protección de rutas, validación cliente-servidor y manejo de tokens.
- **UI/UX Profesional**: Tailwind CSS + componentes reutilizables, mobile-first responsive, loading states y error handling.

**Tech:** React 18 | Next.js 14 | TypeScript | Tailwind CSS | Python/PostgreSQL
